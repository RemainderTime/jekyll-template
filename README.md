# jekyll-template

[See a demo in GitHub Pages](#)

## Installation

```sh
git clone https://github.com/RemainderTime/jekyll-template
```

## Running a local server

```sh
jekyll serve
```

## Deploying it in github

```
git remote add pages <Your github url>
git push pages master
```

## Updating this theme

```
git pull origin
```
